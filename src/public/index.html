Productos

Mortero Español

Mortero español es una mezcla en polvo de adhitivos y cemento que forma un pegante de alta y baja absorción.

Empaste Exterior

Base acrílica hermética de alta calidad para paredes y tumbados exteriores, resistente al agua y rayos UV.

Empaste Interior

Base acrílica antibacteriana, brinda soporte liso en paredes y tumbados interiores previo a la pintura.

Porcelana

Mezcla de base cementicia formulada con pigmentos de colores y adhitivos para juntas de 1 a 7mm. Ideal para la renovación de tus pisos, exteriores e interiores.

Polvo Mineral

Polvo pigmento o color para cemento, ColorMAX. Sirve para cambiar el tono o color de superficies o morteros a base de cemento.

Blanqueador

Pintura cementicida con adhitivos para fondear paredes, ideal para blanquear y enlucir paredes, mamposteria y concretos prefabricados.
